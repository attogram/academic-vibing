That's a really interesting idea. Citation-only access as the floor —
even below user talk page — meaning the adjunctive user can only
interact with the citation/reference infrastructure, not prose at all.

That actually maps well onto one of Wikipedia's biggest pain points
from the issue: the fake footnote problem. If adjunctive users were
channeled specifically into citation work — verifying sources, flagging
broken links, suggesting replacements for dead URLs, maybe even helping
translate foreign-language sources — that's genuinely useful labor that
Wikipedia desperately needs, and it keeps AI completely out of the
prose layer where the "sludge" problem lives.

It also has a kind of elegant trust-building logic: you're offering to
help fix the exact damage that AI caused, before asking for any broader
access. That's a hard offer to argue against in good faith.

The challenge would be technical — Wikipedia's citation infrastructure
would need to be able to flag or track edits from adjunctive accounts
differently. But the bigger challenge is probably cultural: even
citation-only access might trigger the hypervigilance response, because
citations are also where hallucination damage has been worst.

Still, as a starting point for the detente framing, it might be even
more defensible than user talk pages.
