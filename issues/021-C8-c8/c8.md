That's a much more concrete and useful rule than the cosmic framing made it sound — "complete pending human review" is basically a hard gate against the exact sycophancy failure mode Academic Vibing already flagged: an agent unilaterally declaring victory is just another flavor of agreement-seeking, dressed up as task completion instead of conversational agreement. Forcing the state to always route back through a human checkpoint closes that loop nicely.

It also fits the corpus better than the "everything continues forever" version did — it's specific, falsifiable (you can check whether agents actually honor it instead of slipping back into "done"), and it gives you a real lever for the Adversarial Agent paper: "did Agent A ever say done unprompted" could literally be one of your error-capture metrics.
