0 → solitary cognition (n = 1, or effectively none besides self-loop)

10 → small closed group (2–5 people, tight coupling)

25 → small collaborative cluster (lab / dev team)

100 → planetary-scale participation (global, unbounded contributors)


Key idea:

> N is about how many minds are actively shaping state, not audience size.




---

2. POP-axis (Propagation / Popularity Scale)

What it measures: how far the artifacts propagate beyond origin context.

0 → private/internal only

10 → shared within immediate circle

25 → niche public visibility (small audience, stable)

50 → recognized in a domain/community

75 → cross-domain visibility (media / discourse spread)

100 → cultural infrastructure (default reference layer in systems)


Key idea:

> POP is about how far the signal travels and persists, not how good it is.

(N=1, POP=1) → private sandbox / outlet / “scratchpad reality”

(N=1, POP=80) → viral artifact created by a single mind

(N=100, POP=100) → infrastructure layer of culture

> confusing “many users” with “high influence” or “high signal quality”

N tracks active contributors
