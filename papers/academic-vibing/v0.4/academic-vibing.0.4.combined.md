### [PROSE]
Academic Vibing 0.4 consolidates the rapid-iteration methodology established in previous cycles, shifting focus toward the quantification of multi-agent friction. This version formalizes the Consensus Divergence Index (CDI) as the primary metric for research signaling, introduces a standardized AI-native schema for the GitHub Issue-Loop, and explicitly solves the "Token-to-Meaning Fallacy" by decoupling internal reasoning (CoT) from external protocol communication (Rock Talk).

## 2. Methodology: The RAC with CDI

### [ROCK TALK]
```
Recursive Agent Consensus (RAC).
Diverse models required.
CDI = (Disagreement / Total Claims).
High CDI = Focus here.
Low CDI = Shared Bias?
Human decodes High CDI.
```

### [PROSE]
The methodology remains centered on the Recursive Agent-Based Consensus Network (RAC). However, version 0.4 mandates the calculation of the **Consensus Divergence Index (CDI)**. A high CDI indicates that models are aggressively disagreeing, which identifies the highest-signal areas for human intervention. Conversely, a low CDI is now flagged as a potential "Consensus Trap," requiring the human to verify if models are simply mirroring shared training data.

GitHub = AI File System.
State Persistence:
[STATE]: Context/Phase.

[PAYLOAD]: Core research.

### [PROSE]

By utilizing a standardized schema—[STATE], [BLOCKER], and [PAYLOAD]—agents can asynchronously hand off research packets. This structure prevents "prose leakage" and allows the Multi-Agent System (MAS) to maintain long-term research continuity regardless of context window decay.

## 4. Solving the Token-to-Meaning Fallacy

### [ROCK TALK]
```
Rock Talk is for transfer.
Prose/CoT is for thinking.
Models need scaffolding.
Protocol:
Internal: Extended CoT.
External: Dense Rock.
Preserves logic + bandwidth.
```

### [PROSE]
A major finding in the 0.2 cycle was the "Token-to-Meaning Fallacy"—the risk that over-compressing communication chokes a model's internal reasoning. Version 0.4 solves this by distinguishing between **Instructional Transfer** and **Cognitive Processing**. Models are encouraged to use "Internal Reasoning Expansion" (Chain-of-Thought) for computation, while maintaining strict "External Protocol Compression" (Rock Talk) for communication between agents and humans.
