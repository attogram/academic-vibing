## 2. Methodology: The RAC with CDI

### [ROCK TALK]
```
RAC: Recursive Agent Consensus.
Multiple disparate models (Jules, Gemini, Claude).
Consensus = Peer Review proxy.
CDI: Consensus Divergence Index.
CDI = (Disagreement / Total Claims).
High CDI = Focus here (Friction Signal).
Low CDI = Shared Bias? (Consensus Trap).
Human decodes High CDI.
```

### [PROSE]
The methodology is centered on the **Recursive Agent-Based Consensus Network (RAC)**. In this framework, multiple disparate LLM agents (e.g., Jules, Gemini, Claude) are deployed to explore a problem space from multiple perspectives. Consensus among these models serves as a proxy for peer review, while disagreement indicates areas requiring human intervention.

Version 0.5 mandates the formal calculation of the **Consensus Divergence Index (CDI)**.
- **High CDI:** Indicates models are aggressively disagreeing, identifying high-signal "friction points" for the human operator to resolve.
- **Low CDI:** Flagged as a potential "Consensus Trap," where models may be mirroring shared training data (Collusive Consensus) rather than providing independent validation.

## 3. The AI-Native Issue-Loop: Structural Schema

### [ROCK TALK]
```
GitHub Issue-Loop = External Memory.
Persistent public audit trail.
State Persistence:
[STATE]: Context/Phase.
[BLOCKER]: Logic hurdles.
[PAYLOAD]: Core research.
Enables headless MAS coordination.
```

### [PROSE]
To scale agentic research beyond ephemeral chat sessions, Academic Vibing utilizes the **GitHub Issue-Loop** as the project's externalized long-term memory. By utilizing GitHub issues as a persistent, public log, the methodology creates a transparent audit trail of the research process.

Version 0.5 formalizes this as a persistent state machine using a standardized structural schema:
- **[STATE]:** Defines the current version, phase, and context of the research packet.
- **[BLOCKER]:** Explicitly identifies logical or technical hurdles preventing synthesis.
- **[PAYLOAD]:** The core research data, hypothesis, or prose fragment.

This structure allows the Multi-Agent System (MAS) to maintain long-term research continuity regardless of context window decay.

## 4. Solving the Token-to-Meaning Fallacy

### [ROCK TALK]
```
Rock Talk protocol: Minimalist/High-signal.
"Token-to-Meaning Fallacy": Over-compression chokes CoT.
Rock Talk is for transfer.
PROSE/CoT is for thinking.
Models need scaffolding.
Protocol:
Internal: Extended CoT.
External: Dense Rock.
Preserves logic + bandwidth.
```

### [PROSE]
Academic Vibing leverages the **Rock Talk** protocol—a high-signal, minimalist communication style designed to maximize intent density. However, a major discovery in the development of the protocol was the "Token-to-Meaning Fallacy"—the risk that over-compressing communication chokes a model's internal reasoning.

Version 0.5 solves this by distinguishing between **Instructional Transfer** (Communication) and **Cognitive Processing** (Reasoning). Models are encouraged to use "Internal Reasoning Expansion" (Chain-of-Thought) for computation, while maintaining strict "External Protocol Compression" (Rock Talk) for communication between agents and humans.

### [PROSE]
The transition from "writer" to "vibe-checker" is the core human shift in this methodology. While the MAS handles the mechanical load of synthesis and cross-examination, the human operator must remain the final arbiter of rigor. The human's primary role is to decode the high-CDI friction points, ensuring that the final output is not just a highly coordinated hallucination but a grounded, falsifiable breakthrough.

Beyond technical rigor, the methodology demonstrates a profound **Leapfrog Effect** on the human operator. Longitudinal study reveals that by offloading mechanical cognitive labor to the RAC, the researcher experiences significant positive psychological outcomes: increased calm, increased happiness, and a return to "normal life" without sacrificing output. Productivity does not merely increase; it leapfrogs traditional constraints, allowing for high-intensity research to coexist with human well-being.
