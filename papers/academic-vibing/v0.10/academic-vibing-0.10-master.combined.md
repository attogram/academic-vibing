## 2. Methodology: The Consensus Divergence Index (CDI)

### [ROCK TALK]
```
CDI: Friction as Signal.
Formal Definition:
CDI = d / n
d: disputed claims.
n: total claims.
High CDI (>= 0.5): Signal Boundary.
Low CDI (<= 0.1): Consensus Trap.
```

### [PROSE]
The primary metric of Academic Vibing is the **Consensus Divergence Index (CDI)**. The CDI measures the semantic friction between disparate model architectures (e.g., Claude, Gemini, and GPT families) when evaluating a set of research claims.

#### 2.1 Mathematical Formalization
We define the CDI as the ratio of disputed claims to the total number of claims evaluated:

$$CDI = \frac{d}{n}$$

Where:
- $d$ is the number of claims where at least one agent in the swarm disagrees with the majority.
- $n$ is the total number of claims evaluated.

#### 2.2 Interpretation
- **High CDI ($\ge 0.5$):** Indicates a "High-Signal Research Boundary." The models are unable to form a consensus, identifying exactly where the human researcher must focus their investigative energy.
- **Low CDI ($\le 0.1$):** Indicates a "Consensus Trap" or "Shared Bias." The models are in perfect agreement, which in a synthetic environment often suggests they are mirroring shared training data or sycophantic patterns rather than independent validation.

## 3. The Leapfrog Mechanism: A Hypothesis for Re-entry

### [ROCK TALK]
```
Claim #16: Leapfrog Effect.
Hypothesis: Externalized memory reduces re-entry cost.
Standard: 23min recovery (Mark et al. 2008).
AV Goal: < 3min recovery.
Mechanism: Issue-Loop as Exocognitive State Store.
Experimental Design: n=2 longitudinal study.
```

### [PROSE]
The most significant operational claim of Academic Vibing is the **Leapfrog Effect** (Claim #16). We hypothesize that by externalizing research state into a structured, persistent "Issue-Loop," the cognitive cost of task re-entry is significantly reduced.

#### 3.1 The Re-entry Problem
Traditional research on task interruption suggests that human recovery from a deep cognitive break requires an average of 23 minutes <a id="cite-10"></a>[[10]](#ref-10). This "Re-entry Tax" is the primary cause of researcher burnout and "Vibe Hangovers."

#### 3.2 Proposed Experimental Design
To validate the Leapfrog Effect, we propose a longitudinal study comparing:
- **Control Group:** Traditional research workflows where context is maintained in internal working memory and unstructured notes.
- **Test Group (Academic Vibing):** Workflows utilizing the GitHub Issue-Loop as a persistent exocognitive state store.

We predict that the test group will demonstrate re-entry times of less than 3 minutes, effectively "leapfrogging" the traditional interruption recovery period and resulting in a measurable increase in both productivity and subjective wellbeing.

## 4. The AI-Native Issue-Loop

### [ROCK TALK]
```
GitHub = Cognitive Extension.
[STATE], [BLOCKER], [PAYLOAD].
Persistent memory across agent sessions.
Lossless transfer.
```

### [PROSE]
Academic Vibing treats the GitHub repository as a functional extension of the researcher's mind. The **Issue-Loop** provides a structured schema for maintaining research state across disconnected agent sessions:
- **[STATE]:** Current version and phase.
- **[BLOCKER]:** Immediate logic hurdles.
- **[PAYLOAD]:** The research data or prose fragment.

This structure ensures that the research remains "lossless," preventing the decay of context that typically occurs in long-running human-AI collaborations.
