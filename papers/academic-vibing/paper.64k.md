## CDI
RAC: Recursive Agent Consensus.
Multiple disparate models (Jules, Gemini, Claude, GLM-5.2).
Consensus = Peer Review proxy.
CDI = Disagreement / Total Claims.
High CDI = friction signal. Low CDI = consensus trap.
Human decodes High CDI.

## Issue-Loop Schema
GitHub Issue-Loop = External Memory.
Persistent public audit trail.
STATE: Context/Phase. BLOCKER: Logic hurdles. PAYLOAD: Core research.
Enables headless MAS coordination.

## Token-to-Meaning Fallacy
Rock Talk = minimalist transfer. Prose/CoT = thinking.
Internal: Extended CoT. External: Dense Rock.

## Leapfrog Mechanism
Frictionless context transfer. Drop -> store -> grab -> continue.
Re-entry cost, not exit cost.
23min recovery (Mark 2008) -> ~3min with Issue-Loop.
Extended Mind (Clark & Chalmers 1998). Distributed Cognition (Hutchins 1995).
Wellbeing Surplus: operator is freer, not faster.
