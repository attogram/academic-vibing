# Scripts

Automation and analysis tools used in the project.

- `calculate_cdi.py`: Computes the Consensus Divergence Index from multi-model claim sets.
- `generate_podcasts.py`: Generates audio and transcripts for research logs.
- `generate_spectrum.py`: Creates file-size spectrum variants for audio files.
- `generate_rock_mascot.py`: (Archived) Mascot asset generation.
